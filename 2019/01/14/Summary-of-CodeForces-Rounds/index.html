Overview

在 CodeForces 上做的比赛多了，原来的总结方式显得有些乱。

我决定把比赛的记录放到一篇里，简单的记录一下每次的排名和 Rating 变化。

关于题解，以及实现的细节就写到解题报告里了，链接在本篇的最下面。

现在已经建立了 github 上的代码仓库，CodeForces 上的代码都放到 这里 了。

希望自己能坚持住定期更新，有问题请联系我，万分感谢。
