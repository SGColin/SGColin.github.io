OI 生涯就这样恍恍惚惚的在省选走到了终点。

睡了一天一夜幡然醒悟。

回想起来有些感慨，还是决定写一篇流水账，算作是对这段生活的告别。

Memorabilia

初中

我叫 Colin，初中参加过普及组竞赛，是一个 Pascal 选手。

2014 年刚入学的时候，偶然的机会接触到了 OI。

在河北这个几乎没有初中搞竞赛的地方，我在的初中 NOIP 普及组发展的貌似不错。

本着对编程的向往，和还可以看的数学成绩，就去报了名，也没有当作什么正业。

照着书敲下的第一个代码，是一个简单的交互程序：

随机一个数，让你去猜，告诉你猜的结果和正确答案的大小关系。

当时觉得极其满足，幻想着将来能自己想出来一个如此有用的东西。

然后开始了一周只有两个小时会接触 OI 的生活，学习了半年多的 Pascal 语法。

初中的机房里没有网，甚至连题目都是老师写在教案上，然后同学们上去抄。

老师就从网上抄下来整整两本的模拟题，我们也练了整整一年的输入输出和模拟。

然而这一切还是有效果的，至少我在考场上写完了《金币》和《扫雷游戏》。

第一次去往邯郸，参加 NOIP 2015 的普及组，拿到了 $200$ 分省一等奖。

NOIP 2015 回来之后，学了 DFS 和简单的几个 DP 模型，大部分时间还是在写模拟题。

记得做出来《笨小猴》的时候，得知是提高组题目，十分兴奋。

这一年里疯狂的迷恋上了机器人，缠着父母买了 EV3，拉着小伙伴一起乱编指令。

直到最后一次参加普及组，回文的模拟没有写对，这一年只有 $130$ 分省二等奖。

直到去秦皇岛的这一趟，NOIP 2016 分数爆炸，才开始决定离开 OI，回归主线。

NOIP 2016 回来之后，就要面临中考。

一次无心应考的高中招生考试，物理基本交了白卷，最后仔细考虑却交上了保证金。

临近中考的时候，突然被告知，因为 NOIP 成绩，一所高中 OI 强校伸来了橄榄枝。

然而当时想着高中肯定不会继续碰 OI 了，就拒绝了好意，现在看来有些天真。

最后中考的成绩除了体育貌似都还不错，分式也超过了想去的几所学校的正式录取线。

高中

我叫 SGColin，参加过提高组竞赛，是一个 C++ 选手。

入学考试爆炸没进实验班，正在沮丧之余突然有老师联系到我，才知道我在的高中也有信息学竞赛。

刚入学对高考也没有什么概念和对自己的要求，觉得既然能接着学就加入吧，大不了少两三天的假期。

然后就认识了机房里的大家，逐渐知道了从 NOIP 开始往后的路有多漫长。

于是用 Pascal 装作复习了两个月的模拟，就去燕大参加 NOIP 2017 了，最后只有 $60​$ 省三都没混上。

学校组建竞赛班，因为文化课成绩太差，加上不是理科实验班，换班无望。

然后用了将近一个月的时间学了 C++ ，迷迷糊糊的就回去学文化课了，期末考的还很爆炸。

之后一些巧合转进了竞赛班，NOIP 2018 考的也不是很好，最后是 $420$ 省一等奖。

这一年的生活去看 NOIP 2018 和 Goodbye Wuxu 吧，也不是很想再细写了。

出于对 OI 的不甘还是留了下来，也就因此告别了一些曾经在机房共同奋战的朋友。

寒假没能去成 NOIWC，去了 CCPC-Wannafly Winter Camp，认识了 czy 。

然后就来到了 HEOI 2019 的考场，第一天迷糊了三个小时，然后就结束了，第二天没能翻盘。

考试的题目除了 Day1 T3 有些不太好接受以外都还挺好的，最后是被两天 T2 送退役的。

最后的总分是 $85+0+12+20+60+8=185​$，一个啥都不是的成绩，考场降智严重。

看了看题解，感觉扔掉了 $15+100+38+30+40+12=235​$，Day2 T2 放到 NOIP 我可能能做出来。

说说失分的原因吧：

  • Day1 T1：可持久化 01 Trie 的 copy 手残，复制时候的 size 设置写错，最后数据水拿到了 $85​$：

    c[rt].sz=c[lst].sz+1 写成 c[rt].sz=1 ，考场 1h 没找出来心态爆炸，又花了 10min 写了暴力。

  • Day1 T2：把后缀树的性质想成了 SAM 本身DAG 的性质，直接在 SAM 上优化连边以为正解直接爆零。

  • Day1 T3：最后交卷前 5min 才开始当作看玩笑看后面的点，直接看出 p,u 是啥意思但是来不及写。
  • Day2 T1：考场想出来状态设计，想了想转移复杂度以为不对，就放过去了。
  • Day2 T2：想到了正解的思博贪心，觉得左侧多个子节点合并到右侧一个子节点中会挂掉，于是打暴力。
  • Day2 T3：爆 int。

最后反过来想想自己的水平确实没有达到 Day2 T1, Day2 T3 这种题的要求，但是之前的又模拟赛太过简单，考挂或许在情理之中吧。

第二天出考场的时候大概也是释然了吧，出考场的时候几位教练都在紧张的看着我下楼。

我看到了母亲在笑着看着我，我就笑了，老师们也就突然笑了，拍拍肩膀，一切都过去了。
